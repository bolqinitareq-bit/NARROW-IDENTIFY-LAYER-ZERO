# Narrow — Architecture Maps

This folder contains high‑level maps and diagrams describing how Narrow works internally.  
These documents help contributors and partners understand the architecture at a glance.

## Included Files

### 1. architecture-overview.md  
A high‑level map of the Narrow protocol layers.

### 2. data-flow.md  
Explains how identity, schemas, and attestations move through the system.

### 3. components-map.md  
Breakdown of the major components: identity root, schema system, attestation framework, resolution layer, and integrations.
